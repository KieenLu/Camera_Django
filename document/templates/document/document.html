Welcome to Developer API!

HANET API gồm 2 phần chính:

  1. Tập hợp danh sách các HTTP API để developer có thể tương tác với backend
  2. Webhook để chủ động gửi dữ liệu về cho bạn khi có sự kiện phát triên trên camera

HTTP API

Để sử dụng API, trước tiên bạn cần sử dụng OAuth2 để xác thực và ủy quyền

  1. Truy cập vào website http://127.0.0.1:8000/home/API
  2. Đăng nhập hoặc đăng kí tài khoản
  3. Tạo App để lấy client_id và client_secret
  4. Sử dụng thông tin có để lấy access_token theo chuẩn OAuth2
  5. Sử dụng access_token với các API tại đây
Tham khảo về OAuth2:
  • https://www.digitalocean.com/community/tutorials/an-introduction-to-oauth-2
  • https://www.oauth.com/playground/

Webhook

Camera AI cho phép bạn cấu hình webhook để chủ động gửi dữ liệu về hệ thống của bạn khi có sự kiện xảy ra trên camera.

  1. Truy cập vào website http://127.0.0.1:8000/home/API
  2. Đăng nhập hoặc đăng kí tài khoản
  3. Tạo App hoặc chọn App đã có
  4. Tại mục Webhook, điền URL của bạn. Lưu ý: URL webhook của bạn cần phải trả về HTTP Status Code 2xx (200, 201…) khi gọi bằng POST request

Dữ liệu được gửi qua webhook được mô tả chi tiết tại đây
